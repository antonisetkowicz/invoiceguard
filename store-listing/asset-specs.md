# Store Listing Asset Specifications

## App Icon
- Size: 512 x 512 px
- Format: PNG (32-bit, no alpha)
- Design: Dark background (#0A0A0A), centered microphone icon in coral/red (#E94560), minimal style
- Must NOT include text "VoxDrop" in the icon (Google Play policy)

## Feature Graphic
- Size: 1024 x 500 px
- Format: PNG or JPG
- Design suggestion: Dark gradient background, "VoxDrop" in bold white text, tagline "Your voice. One stranger. Zero filters." beneath, abstract waveform lines in coral
